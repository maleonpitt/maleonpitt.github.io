A secure, instructor-facing web app that transforms uploaded assessment spreadsheets into per-student performance summaries and sends individualized feedback emails at scale.

  • Built a React-based instructor portal to upload assessment metadata and student results, then review generated per-student feedback in a clean, scannable UI.
  • Implemented a Flask API to parse Excel inputs, map missed questions to categories/modules, and compute structured summaries (score, focus areas, “needs most work”) for each student.

  • Containerized the system with an Nginx reverse proxy and a production WSGI server, including health checks and secure cookie handling for deployed environments.

Tech: React, Create React App, Axios, Flask, Gunicorn, pandas, openpyxl, python-dotenv, requests, requests-oauthlib (OAuth2), Nginx, Docker, Docker Compose, Let’s Encrypt, OpenAI API (LLM)

  • Docker Compose deployment with Nginx serving the static UI and proxying API/auth routes to the Flask service.
  • TLS termination via Let’s Encrypt certificates mounted at runtime; container health checks for basic uptime monitoring.
